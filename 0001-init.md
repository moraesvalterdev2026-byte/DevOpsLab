# ADR 0001: Inicialização do Projeto e Estrutura Base

**Data:** 2026-07-06

**Status:** Aceito

## Contexto

Este ADR marca a inicialização formal do projeto AXES Bank, estabelecendo a estrutura de diretórios inicial, as convenções de automação (`Makefile`, scripts) e os princípios de governança como código.

## Decisão

Adotar a estrutura de projeto definida no `MasterEngineeringBlueprint.md` e implementar as automações iniciais para Developer Experience (DX) e governança.

## Justificativa

Garantir uma base sólida e organizada para o desenvolvimento futuro, alinhada com as melhores práticas de DevOps e Platform Engineering.
