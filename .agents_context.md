# Contexto do Projeto AXES Bank
- Objetivo: Aplicação Node.js com PostgreSQL, focada em DevOps de Agentes.
- Arquitetura: 
  - Orquestração: Makefile centraliza execução de scripts em `scripts/`.
  - Agentes: Scripts Node.js/Bash que editam artefatos (HTML, CSS, JS, SQL).
  - CI/CD: GitHub Actions com validação de lint/test.
- Regra de Ouro: Nunca sobrescreva arquivos com templates estáticos. Sempre leia o arquivo existente, aplique a modificação solicitada (usando técnicas de edição incremental) e valide com os testes antes de concluir.
- Padrão de Codificação: ES Modules (import/export), async/await, tratamento de erros robusto.
