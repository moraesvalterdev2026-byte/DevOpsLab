# Visão Geral do Projeto: DevOpsLab

## Missão da Plataforma
Criar uma plataforma de aprendizado em Engenharia DevOps que simule um ambiente de produção moderno, escalável e resiliente.

## Portfólio de Serviços
- [ ] Aplicação Web base (Node.js)
- [ ] Infraestrutura como Código (IaC) via Docker/Docker Compose
- [ ] Automatização de deploy (CI/CD)
- [ ] Monitoramento e Observabilidade básica
