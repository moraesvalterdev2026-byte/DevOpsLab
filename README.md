# AXES Bank

Projeto de infraestrutura bancária moderna, construído com foco em escalabilidade, containerização e automação. O AXES Bank utiliza uma arquitetura robusta para garantir disponibilidade e segurança.

## 🚀 Sobre o Projeto
O AXES Bank é uma aplicação Node.js integrada a um banco de dados PostgreSQL. Todo o ambiente foi containerizado para garantir consistência entre os ambientes de desenvolvimento, testes e produção. Atualmente, o projeto está hospedado em uma instância AWS EC2, seguindo as melhores práticas de DevOps.

## 🛠️ Tecnologias Utilizadas
* **Backend:** Node.js
* **Banco de Dados:** PostgreSQL
* **Infraestrutura:** Docker & Docker Compose
* **Cloud:** AWS (EC2)

## ⚙️ Como Executar Localmente
Certifique-se de ter o Docker e o Docker Compose instalados.

1. Clone o repositório:
   `git clone git@github.com:moraesvalterdev2026-byte/DevOpsLab.git`
2. Acesse a pasta do projeto:
   `cd DevOpsLab`
3. Suba o ambiente:
   `docker-compose up --build`
